# koa simple restful starter

一个简洁的koa起手项目

语言: typescript ===> 支持es6模块

支持开发热更新

支持跨域请求

未内置模板引擎

使用sequelize, 默认数据库为mysql

开发时 bash执行 tsc -w && nodemon ./server/main.js

或打开两个命令行, 分别执行以下操作

1 - npm run tsc 

2 - npm run nodemon

部署到服务器，只需要 server 目录

启动执行 npm run start

关闭所有nodejs后台服务 npm run stop

命令详见package.json
